quark-domready 0.1.1
====================

[dom-ready](https://github.com/noordawod/dom-ready) module for [quark](https://github.com/pyrsmk/quark).

Install
-------

```shell
npm install quark-domready
```

Use
---

```js
$(function() {
	// the DOM is ready
});
```
